# Display Multiple Plots Simultaneously from ggExametrika Output"

This function consolidates multiple plots generated from the
ggExametrika function into a single display.

## Usage

``` r
combinePlots_gg(plots, selectPlots = c(1:6))
```

## Arguments

- plots:

  ggExametrika output plots

- selectPlots:

  In the vector, specify the range or indices you want to display.
