# Plot Class Membership Profile (CMP) from Exametrika

This function takes Exametrika output as input and generates Class
Membership Profile (CMP) using ggplot2. The applicable analytical
methods are Latent Class Analysis (LCA) and Bicluster Network Model
(BINET).

## Usage

``` r
plotCMP_gg(data)
```

## Arguments

- data:

  Exametrika output results
