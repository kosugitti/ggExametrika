# Plot FieldPIRP from Exametrika

This function takes Exametrika output as input and generates FieldPIRP
using ggplot2. The applicable analytical methods is Local Dependence
Biclustering (LDB). The warning message regarding NA values is
displayed, but the behavior is normal.

## Usage

``` r
plotFieldPIRP_gg(data)
```

## Arguments

- data:

  Exametrika output results
