# Plot Field Reference Profile (FRP) from Exametrika

This function takes Exametrika output as input and generates Field
Reference Profile (FRP) using ggplot2. The applicable analytical methods
are Biclustering, Infinite Relational Model (IRM), Local Dependence
Biclustering (LDB), and Bicluster Network Model (BINET).

## Usage

``` r
plotFRP_gg(data)
```

## Arguments

- data:

  Exametrika output results
